Steps to run the project:-
1. Open the Python File
Locate and open the currency converter .py file in your preferred editor or IDE (IDLE, VS Code, PyCharm, etc.).

2. Run the Program
Execute the script by pressing Run or using the command:
python filename.py

3. Choose the Source Currency
The program will ask you to enter the currency you want to convert from (e.g., USD, INR, EUR).

4. Choose the Target Currency
Next, enter the currency you want to convert to.

5. Enter the Amount
Type the amount of money you want to convert.

6. View the Converted Value
The program calculates and displays the converted amount based on predefined exchange rates.
